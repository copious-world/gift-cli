load_config_file(conf_file) → {object|boolean}

Attempts to load the JSON formatted configuration file, parse it and return it. If it can't load the file, it will report an error message and return false The error message is custom for use in the cli. It expects users will provide `load conf` as a command. tokenize(line) → {Array.<Token>}

Takes in single line of text and passes it to jsTokens for parsing. Calls `clump_clumps` to put file paths and similar strings into single atoms. Removes white spaces from the list as well as tokens that have been marked for discard by `clump_clumps`.
